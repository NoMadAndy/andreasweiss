📄 Kampagnen-Flyer erstellen

Erstelle ein druckbares PDF-Dokument zum Verteilen an Haushalte. Alle Inhalte werden aus deinem Profil geladen und können hier angepasst werden.

Wähle eine Unterseite, um einen themenspezifischen Flyer zu erstellen. Headline, Text und QR-Code werden automatisch angepasst.

Die URL wird als grosser QR-Code auf dem Flyer dargestellt.

Vorschlag fuer laengeren Flyer-Text: Nutze Stichpunkte (je Zeile mit - beginnen), um deine Kernbotschaften und den Mehrwert der Seite hervorzuheben. Die Punkte erscheinen als Aufzaehlung im PDF.
